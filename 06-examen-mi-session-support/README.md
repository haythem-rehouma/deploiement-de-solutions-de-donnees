# 06 - Examen mi-session : support formatif

Ce dossier contient un cours de référence et des examens formatifs pour préparer l'examen de mi-session.

## Contenu

| Fichier | Sujet | Description |
|---------|-------|-------------|
| **P00-Cours-Complet-Jenkins-Kubernetes.md** | Cours complet | Référence exhaustive couvrant DevOps, CI/CD, stratégies de déploiement (Rolling Update, Blue/Green, Canary, A/B Testing, Shadow), Git, Jenkins, Groovy, Kubernetes, Services, Kind. À lire avant les examens. |
| P01-Jenkins-Fondamentaux-Examen.md | Jenkins : fondamentaux | QCM, Vrai/Faux, questions courtes, complétion de Jenkinsfile (25 questions, /30 pts) |
| P02-Jenkins-Pipeline-Groovy-Exercices.md | Jenkins : pipelines Groovy | Exercices de complétion de pipelines (12 exercices, /30 pts) |
| P03-Kubernetes-Bases-Quiz.md | Kubernetes : bases | Concepts, commandes, Pods, Deployments (30 questions, /30 pts) |
| P04-Kubernetes-Services-Avance-Quiz.md | Kubernetes : avancé | Services, Kind, namespaces, labels, QCM (30 questions, /30 pts) |
| **P05-Examen-Blanc-Complet.md** | **Examen blanc (sans corrige)** | **54 questions couvrant tous les modules + CI/CD + strategies de deploiement, sans reponses (90 min, /80 pts)** |

## Comment utiliser ce dossier

1. Lire **P00** en entier pour réviser toutes les notions.
2. Faire les examens **P01 à P04** sans regarder les réponses.
3. Vérifier ses réponses en cliquant sur "Voir la réponse".
4. Relire les sections de P00 correspondant aux questions ratées.
5. Faire **P05** (examen blanc sans corrigé) en conditions réelles pour s'auto-évaluer.

## Format

- Toutes les réponses sont cachées avec `<details>` (cliquer pour révéler).
- Chaque fichier contient un barème d'auto-évaluation.
- Les examens sont conçus pour être réalisés en 30 à 45 minutes chacun.
