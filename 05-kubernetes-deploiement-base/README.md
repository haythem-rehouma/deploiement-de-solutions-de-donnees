# Module 5 : Kubernetes - Déploiement de Base

## Vue d'ensemble

Ce module est une **introduction douce** à Kubernetes pour les débutants. Nous allons découvrir les concepts fondamentaux sans entrer dans les détails complexes.

**Prérequis :** Avoir complété le module 04 (Docker)

---

## Structure des cours

| Cours | Titre | Durée | Contenu |
|-------|-------|-------|---------|
| 01 | C'est quoi Kubernetes ? | 30 min | Introduction, vocabulaire de base |
| 02 | Installation avec Minikube | 30 min | Environnement local |
| 03 | Les Pods | 45 min | L'unité de base de Kubernetes |
| 04 | Les Deployments | 45 min | Gérer des applications |
| 05 | Les Services | 45 min | Exposer les applications |
| 06 | Projet pratique | 60 min | Déployer une application complète |

---

## Ce que vous allez apprendre

A la fin de ce module, vous saurez :

1. Expliquer ce qu'est Kubernetes et pourquoi l'utiliser
2. Installer un environnement Kubernetes local
3. Créer et gérer des Pods
4. Déployer des applications avec des Deployments
5. Exposer des applications avec des Services
6. Utiliser les commandes kubectl de base

---

## Ce que nous n'abordons PAS (pour l'instant)

Ces sujets sont pour les modules avancés :

- Ingress et routage avancé
- Volumes persistants
- ConfigMaps et Secrets
- Namespaces
- RBAC et sécurité
- Helm
- Opérateurs
