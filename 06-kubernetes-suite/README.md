# Module 06 - Kubernetes Suite

## Objectifs du module

Ce module approfondit les concepts Kubernetes introduits dans le module 05. Il couvre l'installation, la gestion des pods, les deploiements, et les services Kubernetes.

## Prerequis

- Avoir complete le module `05-kubernetes-deploiement-base`
- Connaitre les bases de Docker
- Disposer d'un environnement Linux ou Windows avec droits administrateur

## Structure du module

### Partie 1 : Theorie (Chapitres 1-2)

| Chapitre | Contenu | Duree estimee |
|----------|---------|---------------|
| 01 | Introduction a Kubernetes et orchestration | 30 min |
| 02 | Objets fondamentaux : Pods, ReplicaSets, Deployments | 45 min |

### Partie 2 : Installation (Chapitres 3-4)

| Chapitre | Contenu | Duree estimee |
|----------|---------|---------------|
| 03 | Installation Minikube sur Windows | 45 min |
| 04 | Installation Minikube sur Linux | 30 min |

### Partie 3 : Pratiques (Chapitres 5-9)

| Chapitre | Contenu | Duree estimee |
|----------|---------|---------------|
| 05 | Scaling sans fichier YAML | 45 min |
| 06 | Gestion des Pods | 30 min |
| 07 | Gestion des Deployments avec YAML | 45 min |
| 08 | Kind : Kubernetes in Docker | 60 min |
| 09 | Nettoyage de contexte et nouveau cluster | 20 min |

### Partie 4 : Services Kubernetes (Chapitres 10-13)

| Chapitre | Contenu | Duree estimee |
|----------|---------|---------------|
| 10 | Introduction aux Services | 45 min |
| 11-12 | Demos pratiques des Services | 90 min |
| 13 | Types de Services en detail | 60 min |

## Parcours recommande

```
Theorie (01-02) -> Installation (03 ou 04) -> Pratiques (05-09) -> Services (10-13)
```
